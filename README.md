# Rock Paper Scissors

A simple Python implementation of the classic Rock Paper Scissors game where you play against the computer.

## How to Play
1. Run the script
2. Enter "Rock", "Paper", or "Scissor"
3. Computer randomly chooses
4. See who wins based on standard rules
